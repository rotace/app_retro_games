# Data Model: WSLgでのノイズ描画機能

## 1. エンティティ定義

### NoiseGenerator (コアロジック)
- **役割**: 座標に基づいたノイズ値を計算し、ピクセルデータに変換する。
- **プロパティ**:
    - `seed`: ノイズの種（固定値）。
    - `frequency`: ノイズの周波数（固定値）。

### RenderTarget (描画ターゲット)
- **役割**: 計算されたピクセルデータを実際のディスプレイ（WSLgウィンドウ）に反映させる。
- **プロパティ**:
    - `width`: キャンバスの幅。
    - `height`: キャンバスの高さ。
    - `buffer`: ピクセルデータの配列 (`Vec<u32>`)。

## 2. データフロー
1. `runner-wslg` が `RenderTarget` (バッファ) を用意。
2. `core::noise` が `RenderTarget` のバッファに対して、パーリンノイズに基づく色値を書き込む。
3. `runner-wslg` が `minifb` を介してバッファを画面に表示。

## 3. 制約
- パラメータはすべて固定値とし、ユーザーによる動的変更はサポートしない（仕様に基づく）。
